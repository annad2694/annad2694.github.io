Murder Mystery CYOA Project

Project Context

This project was assigned by our teacher, Mr. Mueller, who wanted us to learn how to work as a team with someone we hadn't worked with before this semester. He aimed to assess our communication and teamwork skills, teaching us how to introduce ourselves to new people and collaborate effectively. However, this project required more than just a partnership; it also demanded a solid understanding of the Git and GitHub concepts we covered earlier. The project gave us an opportunity to practice everything we had learned and ensure we were confident in our skills. We practiced linking files, creating repositories, and learning how to push and pull changes as a team, ensuring we didn't disrupt any progress that had already been committed.

Challenges

A challenge my partner and I faced was our differing preferences. I prefer doing most of my work on GitHub, while Alex preferred working entirely in his IDE. At first, this wasn’t an issue, but it eventually became confusing. I didn’t fully understand how to create links in the IDE because I’m more used to coding directly on GitHub. However, since we worked so well together, Alex showed me different ways the IDE could be used and how it syncs with GitHub. He also explained that editing directly on GitHub creates unnecessary commits, whereas in the IDE, you only need to commit once when you're finished. This became a problem for me, as I would sometimes forget to commit at the end of class, and some of our progress would be lost. To address this, I introduced Alex to GitHub and showed him how to edit there. I explained how the links are already in the correct format and can be copied directly. Since he was creating the links, they often didn’t lead anywhere. It was also easier to view the example Mr. Mueller provided on GitHub, so we kept that tab open on our screen as a reference. Eventually, we both adapted to each other's preferred coding methods, which demonstrated our ability to embrace diversity. We also felt comfortable asking each other questions when we didn’t understand something, as we were always willing to help each other—showing a truly healthy partnership.

Next Steps

There’s nothing I would really change, except perhaps adding pictures. When I viewed the examples Mr. Mueller provided, they featured cool backgrounds with images that matched the topic. If my partner and I had more time, we would have added our own touch by styling the page with different fonts, colors, and images to truly capture the horror theme that was meant to be demonstrated in this project. Next time I feel that I’ve completed a project, I’ll focus on making it more presentable and engaging for the viewer who interacts with my creation.
